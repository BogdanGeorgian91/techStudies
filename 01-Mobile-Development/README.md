# Mobile Development

Comprehensive resources for mobile application development, covering both cross-platform (React Native) and native iOS development.

## Contents

### React-Native/
Cross-platform mobile development using React Native framework.

**Key Topics:**
- **Architecture** - New Architecture, JSI, Turbo Modules, Bridge communication
- **Performance** - Optimization techniques, profiling, memory management
- **Platform-Specific** - iOS and Android native integration
- **Security** - Best practices, authentication, secure storage
- **State Management** - Redux, Context API, TanStack Query
- **Navigation** - React Navigation patterns and deep linking
- **Native Modules** - Creating custom native modules
- **AR** - Augmented reality with Expo XR
- **Features** - Background tasks, geolocation, paywalls, appearance API
- **Tips** - Practical advice from real-world app development

**File Count:** 68 documents

### iOS-Native/
Native iOS development with Swift, SwiftUI, and Apple frameworks.

**Key Topics:**
- **Apple Docs** - Comprehensive Apple documentation
  - CloudKit - Cloud storage and sync
  - Core Data - Local database
  - App Intents - Siri and system integration
  - WidgetKit - Home screen widgets
- **Concepts** - iOS-specific patterns (@Published, ObservableObject, NSPredicate)
- **CoreData and CloudKit articles** - Integration guides and best practices

**File Count:** 99 documents

## Learning Path

### Beginner
1. Start with React-Native/Architecture for understanding how React Native works
2. Explore React-Native/Platform-Specific/iOS or Android basics
3. Learn iOS-Native/Concepts for understanding native iOS patterns

### Intermediate
1. Deep dive into React-Native/Performance optimization
2. Study React-Native/State-Management patterns
3. Explore iOS-Native/Apple Docs/CloudKit for cloud integration

### Advanced
1. React-Native/Native-Modules for creating custom native code
2. React-Native/AR for augmented reality features
3. iOS-Native/Apple Docs for comprehensive native capabilities
